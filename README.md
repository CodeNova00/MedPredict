# MedPredict

AI-powered healthcare platform for disease prediction, skin disease detection, and medical chatbot assistance.

## Features
- Symptom-based disease prediction
- Skin disease image detection
- AI medical chatbot
- Severity and risk assessment
- Doctor appointment booking

## Tech Stack
- Python
- FastAPI
- XGBoost
- Random Forest
- TensorFlow / Keras
- MobileNetV2
- HTML, CSS, JavaScript

## Performance
- 150+ disease predictions
- ~66.4% accuracy in disease prediction
- ~90% skin disease model accuracy
- Real-time prediction support

## Installation

```bash
git clone https://github.com/CodeNova00/MedPredict.git
cd MedPredict
pip install -r requirements.txt
python app.py
```
